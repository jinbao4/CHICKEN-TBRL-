# CHICKEN (TBRL)

CHICKEN is a small, focused anti-abuse plugin for competitive Minecraft servers.  
It fixes broken mechanics without messing with normal gameplay.

---

## Features

### Mace Smash Limiter
- Caps the fall distance used for mace damage
- Stops ridiculous one-shot smash kills
- Optional critical hit scaling
- Fully configurable damage phases
- Keeps maces usable without letting them be stupid

### Egaps Limiter
- Limits how many enchanted golden apples a player can carry
- Prevents bundle-based egap exploits
- Enforced on:
  - Pickup
  - Join
  - Inventory actions
- No easy bypasses

---

## Installation

1. Download the latest release  
   https://github.com/jinbao4/CHICKEN-TBRL-/releases/latest
2. Drop the JAR into your `plugins/` folder
3. Start the server once to generate files
4. Edit `plugins/Chicken/config.yml`
5. Restart or reload

Done.
