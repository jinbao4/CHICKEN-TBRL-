# CHICKEN (TBRL)

- **Mace Smash Limiter**
  - Caps effective fall distance for mace damage
  - Prevents extreme one-shot smash abuse
  - Optional crit scaling
  - Fully configurable damage phases

- **Egaps Limiter**
  - Limits how many egaps a player can hold
  - Blocks bundle-based egap exploits
  - Enforced on pickup, join, and inventory actions

---

## Installation

1. Download the latest release
2. Drop the JAR into your server’s `plugins/` folder
3. Start the server once
4. Edit `plugins/Chicken/config.yml`
5. Restart or reload

That’s it.
